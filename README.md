# Flight Booking Application

A simple **Flight Booking System** built using **Spring Boot**, **MySQL**, and **JPA**.  
This project demonstrates:

- **REST API design**  
- **Database modeling**  
- **Input validations**
- **Request/response handling**
- **Basic unit testing**
